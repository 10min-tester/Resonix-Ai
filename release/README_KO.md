# Resonix AI v0.9.0 for Windows

Resonix AI는 로컬 PC에서 실행되는 AI 기반 오디오 복원/마스터링 도구입니다.
음원을 분석한 뒤 선택한 청감 목표와 AI 적용량에 맞춰 노이즈, 음량, 헤드룸, 스테레오 보존, 음색 보정을 적용합니다.

## 실행 방법

1. ZIP 파일을 원하는 폴더에 압축 해제합니다.
2. 압축 해제된 `ResonixAI-Windows` 폴더 안의 `ResonixAI.exe`를 더블 클릭합니다.
3. 잠시 기다리면 기본 브라우저가 자동으로 열립니다.
4. 브라우저가 열리지 않으면 콘솔 창에 표시된 주소를 직접 입력합니다.

```text
http://127.0.0.1:8000/
```

8000번 포트가 이미 사용 중이면 앱이 8001 이후 포트를 자동으로 찾습니다.

## 사용 방법

1. `오디오 업로드` 영역에 wav, mp3, m4a, flac 파일을 넣습니다. 여러 파일을 한 번에 넣을 수도 있습니다.
2. 청감 목표를 선택합니다. 여러 목표를 동시에 선택할 수 있습니다.
3. `AI 적용량`은 0%에 가까울수록 원본 보존, 100%에 가까울수록 추천 처리값을 강하게 적용합니다.
4. `원본 음량에 맞춤`을 선택하면 처리 후 체감 볼륨이 원본과 비슷하게 맞춰집니다.
5. 단일 파일은 향상된 WAV 파일을 다운로드합니다.
6. 여러 파일은 향상된 WAV 파일들과 `batch_report.json`이 들어 있는 ZIP으로 다운로드합니다.
7. 배치 결과 목록에서 파일을 선택하면 해당 음원을 단일 처리 화면처럼 원본/향상 파형으로 A/B 청취할 수 있습니다.

## 고급 설정

- 출력 샘플레이트: 자동, 원본 유지, 44.1 kHz, 48 kHz, 96 kHz 중 선택할 수 있습니다.
- 출력 비트 깊이: 16-bit PCM 또는 24-bit PCM을 선택할 수 있습니다.
- 수동 DSP override를 켜면 AI 추천값 대신 저역 컷, 저역/중역/고역, 컴프레션, 목표 LUFS, 익사이터, 새추레이션 값을 직접 지정할 수 있습니다.

## 출력 품질

- 출력 파일은 WAV 24-bit PCM으로 저장됩니다.
- 입력 샘플레이트가 낮은 파일은 필요 시 44.1 kHz로 올려 저장합니다.
- 트루 피크 기반 헤드룸을 적용해 클리핑 위험을 줄입니다.
- 스테레오 파일은 미드/사이드 기반으로 처리해 스테이징 손상을 줄입니다.
- 원본과 향상본의 파형을 함께 표시해 처리 전후를 쉽게 비교할 수 있습니다.
- 여러 파일을 같은 AI 설정으로 순차 처리하고 ZIP으로 받을 수 있습니다.
- 배치 처리 후 개별 파일을 선택해 바로 재생하고 개별 향상 WAV도 받을 수 있습니다.
- 분석 리포트에서 출력 포맷, 전후 LUFS, 트루 피크, 스테레오/위상, 적용 단계를 확인할 수 있습니다.
- 전체 트랙 처리는 기본적으로 빠른 노이즈 처리 경로를 사용해 응답성을 우선합니다.

## 종료 방법

실행 중인 콘솔 창을 닫으면 Resonix AI 서버가 종료됩니다.

## 참고

- 이 배포본은 one-folder 방식입니다. `ResonixAI.exe` 주변의 `_internal` 폴더를 지우거나 옮기면 실행되지 않습니다.
- 첫 실행은 모델과 라이브러리 초기화 때문에 시간이 조금 걸릴 수 있습니다.
- Windows SmartScreen 경고가 나타날 수 있습니다. 아직 코드 서명 인증서를 적용하지 않았기 때문입니다.
- 처리 결과와 임시 파일은 사용자 폴더 아래 `.packaged_audio_ai`에 저장됩니다.
- 로그 파일은 사용자 폴더 아래 `.packaged_audio_ai/logs/app.log`에 저장됩니다.
- 이 배포본은 인터넷 연결 없이도 기본 처리 기능을 사용할 수 있습니다.
